Samba client
============


How to connect to samba server from UI on Linux Mint
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. Click Super
2. Open ``Connect to``
3. Fill
    - Server: just your server ip
    - Type: Windows share
    - Folder: /md0
    - Domain name: WORKGROUP
    - User name: seba_nas
    - Password: <HERE_YOUR_PASSWORD>
4. Connect
