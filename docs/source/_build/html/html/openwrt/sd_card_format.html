SD card formatÔÉÅ

If there is lack of some information check this nice tutorial

Steps:

  1. Before start, refresh the list of available software packages

opkg update

  1. Install all utils if you want to verify usb drivers

opkg install kmod-usb-storage
opkg install kmod-usb-storage-uas
opkg install usbutils

  1. List your connected devices

lsusb -t

  1. Check current file system

df -Th

  1. If card exist /dev/sda*/ e.g. /dev/sda1/ try to unmount:

umount /dev/sda1

Otherwise check which one is it

ls -l /dev/sd*

  1. Before format partition, install needed packages

opkg install e2fsprogs
opkg install kmod-fs-ext4

  1. Format with EXT4 file system

mkfs.ext4 /dev/sda1

  1. Mount sd card

mount /dev/sda1 /mnt/sdcard
