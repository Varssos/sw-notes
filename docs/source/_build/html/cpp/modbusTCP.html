Modbus TCP

Simulator

To start developing Modbus TCP client you can use pyModbusTCP to set Modbus TCP server. Below are described few steps how to set it. Connection parameters are hardcoded in python script

  1. Install python3 package sudo pip3 install pyModbusTCP

  2. Download simple modbusTCP server script modbusTCPserver.py

  3. Make file executable for user chmod +x modbusTCPserver.py

  4. ./modbusTCPserver.py

Libraries

For embedded development purposes libmodbus is good solution.

  • written in C

  • LGPL license

  • available on many platforms like OpenWrt without any cross compiling

  • if it is needed it can be easily compiled

  • very popular and well documented
